简易计算器

0.前言

之前写过一个Biginteger类 ，现在学了点C++11和QT就寻思着用它写一个带GUI的计算器。

1.简介

基本用法：在“算式”框中输入表达式，点击“计算”即可在“结果”框中获得结果。输入错误算式将得到错误信息。

功能及注意事项：

  • 目前仅支持加减乘除，不支持括号，且除第一个数外不允许为负数。
  • 除法为整除。如7/3=2,5/3=1
  • 在满足算式总长度不超过215 (32768)的前提下，原则上参与运算的数字可以有任意大小。
  • 支持复制结果到剪贴板。
  • 支持表达式改变时自动计算。

2.快捷键

F1:打开帮助

Esc:清空

Enter:等于
